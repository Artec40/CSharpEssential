Создайте программу, которая будет выводить на экран цепочки падающих символов.
Длина каждой цепочки задаётся случайно. Первый символ цепочки - белый, 
второй символ - светло-зелёный, остальные символы тёмно-зеленые.
Во время падения цепочки, на каждом шаге, все символы меняют своё значение.
Дойдя до конца, цепочка исчезает и сверху формируется новая цепочка.
Расширьте задание так, чтобы в одном столбце одновременно могло быть 
две цепочки символов.
